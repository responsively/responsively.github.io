responsively.js
what

responsively.js is a small, open-source JavaScript micro-library designed to help solve responsive design problems.

It is designed to work well with sites that use client-side templates (handlebars.js, mustache, ...). It is not intended to replace more traditional responsive design techniques (media queries, grids, ...), but rather it should be used as a toolbox to solve specific responsive design problems that are not easily handled using traditional techniques or to augment them -- responsively.js is not a framework.
